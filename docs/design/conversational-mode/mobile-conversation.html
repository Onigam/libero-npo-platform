Principes tactiles
Design optimise touch : boutons > 44px, espacement genereux, navigation par onglets
  • Zones tactiles minimum 44x44px (recommandation Apple HIG)
  • Espacement vertical de 16px entre les messages
  • Chips de reponse rapide accessibles au pouce
  • Navigation par onglets en bas de l'ecran
Conversation first
Sur mobile, la conversation devient le mode d'interaction principal. L'utilisateur pose des questions naturelles et obtient des donnees contextuelles.
  • Input vocal natif pour dictee rapide
  • Cartes de donnees compactes inline
  • Raccourcis thematiques en chips horizontaux
  • Notifications via onglet Alertes (badge rouge)
Navigation simplifiee
4 onglets couvrent l'essentiel : conversation, donnees structurees, alertes et parametres. Pas de sidebar ni de menus complexes.
Pattern UX
Mobile-first -- la conversation est le mode d'interaction principal. L'interface tactile privilegied les echanges naturels en langage libre, les reponses rapides et les donnees condensees adaptees a un ecran reduit.
