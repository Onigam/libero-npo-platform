Description des roles

Admin
Administrateur
Acces complet a toutes les fonctionnalites. Gere les utilisateurs, la facturation, les parametres de l'organisation et les donnees RGPD.
Collecte
Gestionnaire collecte
Gere les campagnes de collecte de fonds, les dons, les constituants et les recus fiscaux. Pas d'acces aux parametres systeme.
Subventions
Gestionnaire subventions
Gere les demandes de subventions, le suivi des financements institutionnels et les rapports bailleurs. Acces aux constituants en lecture.
Comptable
Comptable
Acces aux rapports financiers, exports comptables et integration avec les logiciels de comptabilite. Lecture seule sur les dons et campagnes.
Lecteur
Lecteur seul
Consultation uniquement. Peut voir les tableaux de bord, les rapports et les donnees mais ne peut rien modifier.
